# Vox – AI Voice Assistant

A browser-based voice assistant built by Rich Marra.

## Live Demo

[voice-assistant-kappa-two.vercel.app](https://voice-assistant-kappa-two.vercel.app)

## What It Does

- Voice input via Web Speech API
- Claude Sonnet 4.6 processes the command via secure serverless backend
- ElevenLabs TTS speaks the response in a premium AI voice
- Four selectable voices: Brian, Daniel, Charlie, Bella
- Command chips for quick access without speaking
- Stop Speaking button for immediate playback control
- Light theme UI with navy header and RichBuildsAI.dev branding

## Tech Stack

- Frontend: Vanilla HTML/CSS/JS
- Backend: Vercel serverless function (Node.js)
- AI: Anthropic Claude Sonnet 4.6
- Voice: ElevenLabs Text-to-Speech (eleven_turbo_v2)

## Deploy to Vercel

1. Push this repo to GitHub
2. Import to Vercel
3. Add environment variables: ANTHROPIC_API_KEY and ELEVENLABS_API_KEY
4. Deploy
