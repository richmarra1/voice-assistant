# Voice Assistant

A voice-activated AI assistant built by Rich Marra as part of the 60-in-60 AI portfolio challenge.

## What it does
- Voice input via Web Speech API
- Claude Sonnet 4.6 processes the command via secure backend
- ElevenLabs speaks the response back in a premium AI voice
- Command chips for quick access without speaking

## Tech Stack
- Frontend: Vanilla HTML/CSS/JS
- Backend: Vercel serverless function (Node.js)
- AI: Anthropic Claude Sonnet 4.6
- Voice: ElevenLabs Text-to-Speech

## Deploy to Vercel
1. Push this repo to GitHub
2. Import to Vercel
3. Add environment variable: ANTHROPIC_API_KEY
4. Deploy
